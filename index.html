How do I get an API key?
  1. Go to console.anthropic.com and sign up or log in
  2. Navigate to Settings → API Keys in the sidebar
  3. Click Create Key and give it a name
  4. Copy the key (starts with sk-ant-api03-...) and paste it above

New accounts get free credits to start. This app uses Claude Sonnet which costs roughly $0.01 per set of flashcards generated.
